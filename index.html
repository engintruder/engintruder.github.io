Raspberry Pi & Leap motion

서버 / 클라이언트 형태의 Leap motion
1. Youtube :
2. Servos 예제
3. Leap motion 홈페이지에 있는...
이 내용을 한번 자세히 봐보세요
1번 영상의 경우 튜토리얼이 8인가 9까지 있는데, 자세히 보면 서버 클라이언트 형태로 구동되는것을 확인해볼 수 있구요
2번 페이지도 중간 그림에 PubNub를 보면 노트북에 Leap motion이 설치되어 있고 Raspberry Pi에 Servos가 설치되어 있어요
3번도 2번과 비슷한 것이므로... 구동되는 건 동일할껀데 중간 내용 보면 "Windows desktop running an http server in unity to proxy the data" 라고 써있어요.. 그러면서 아래 내용 보면 Raspberry pi 는 http 서버에서 데이터를 폴링해 leap data 와 일치하는 형태라는 말이 써있는거같네요


Raspberry pi에... Leap motion을 실질적으로 구동시키려는 노력(?)을 하고 있지만 실제로는 안되어 있는 상황 또는 앞으로 해야되겠지만... 아직은 불가능한 상황.. 등등 이라고 얘기하고 있는 트위터 및 포럼 글이예요
트위터 글
립모션 포럼
근데 번외로... 만약에 립모션 드라이브를 라즈베리에다가 설치가 가능하다면~ 아래 내용은 확인해볼만 하겠네요
1. get start sdk: https://developer-archive.leapmotion.com/documentation/javascript/index.html
2. github : https://github.com/leapmotion/leapjs
이게 무엇이냐면 립 모션을 자바스크립트를 통해 컨트롤할 수 있는 버전인데....
이 부분은 라즈베리 파이에 만약 립 모션 드라이브를 설치하고 인식을 했다면 자바스크립트 라이브러리를 통해
구동시킬 수 있다고 전 생각을 하고 있어요
